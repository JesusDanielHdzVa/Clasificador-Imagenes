Esto es el Clasificador de imágenes

El Clasificador de objetos es una herramienta avanzada que utiliza técnicas de aprendizaje automático para analizar y categorizar imágenes. Con solo subir una imagen, el sistema procesa la información visual y determina a qué categoría pertenece, ya sea vegetación, urbanización, piel, madera o roca, proporcionando resultados precisos y rápidos.

¿Cómo Funciona el Clasificador de imagenes?

Nuestro proyecto funciona con distintas tecnologías, tanto para la web como para el análisis de datos. Sin embargo esta misma el analisis de datos fue la principal ya que nuestro proyecto funciona medianta un arbol de decisiones el cual fue entrenado con distintas imagenes las cuales fueron analizadas y etiquetadas con sus respectivas clases, intensidad y contraste.

El proceso de clasificación de imágenes se lleva a cabo en varios pasos clave:

Subida de Imagen: El usuario sube una imagen a través de la interfaz web.
Preprocesamiento: La imagen se procesa para extraer características relevantes como intensidad de color y contraste.
Clasificación: Estas características se ingresan en un modelo de árbol de decisiones previamente entrenado, que analiza los datos y determina la categoría a la que pertenece la imagen.
Resultado: Finalmente, el sistema muestra al usuario la categoría predicha junto con las características analizadas.
